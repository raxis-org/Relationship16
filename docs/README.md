# RelationCheck 16 - ドキュメント

このフォルダには、RelationCheck 16（16タイプ関係性診断Webアプリ）の開発に関するドキュメントが格納されています。

## ドキュメント構成

| ファイル | 内容 |
|---------|------|
| `README.md` | このファイル - ドキュメント全体の説明 |
| `feedback.md` | ユーザー・ステークホルダーからのフィードバック記録 |
| `todo.md` | 未実装項目・タスク一覧 |
| `specs.md` | 仕様書・設計ドキュメント |
